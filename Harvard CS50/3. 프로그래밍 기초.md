# Topic2: 프로그래밍의 기본 문장을 출력할 수 있을까?

## 핵심 단어
- C언어
- 문법
- 함수
- printf
- 컴파일
- 주석

## 수업 내용
### 1. C언어의 문법

- C언어: 프로그램을 작성하는 데 사용되는 프로그래밍 언어
- 문법: 완벽한 프로그램으로 동작하는 문장들을 작성하기 위해 필요한 단어와 기호들(소괄호, 중괄호)을 어떻게 배열할 것인지에 대한 규칙

### 2. 첫번째 C 프로그램
```c
#include <stdio.h>          // 1행

int main(void) {            // 3행
	printf("hello world\n"); 
}
```

- 1행
    - `#include <stdio.h>`  는 이 프로그램이 stdio.h라는 파일 안에 들어있는 함수들에 접근 가능하게 해준다.
    - 이로 인해 화면에 문자를 보여주는 역할을 하는 `printf` 함수를 가져다 사용할 수 있다.
- 3행
    - `int main(void)` 는 프로그램의 시작점을 정의한다.
    - `printf` 는 문자열을 화면에 출력하는 함수이다.
    - `\n` 문자는 줄 바꿈 문자로, 문자열을 출력한 후 줄을 바꿔준다.

### 3. 컴파일하여 프로그램 실행하기
- C 프로그램 작성 후, c로 끝나는 파일로 소스코드를 저장해야 한다.   
이렇게 저장한 소스코드를 0과 1로 만드는 작업인 **컴파일(compile)** 이 필요하다.
    - 컴파일: 소스 코드를 일련의 0과 1들로 이루어진 오브젝트 코드(object code)로 전환하는 것

### 2. 변수를 선언하여 사용하기

1. 변수 선언(declaration): C에서 변수 선언을 해주기 위해서는 변수의 자료형을 알려준 후 변수의 이름을 명시한다.
2. count에 2라는 값을 할당한다.
3. count의 값을 8로 변경한다.

### 3. 사용자 입력을 받는 변수
```c
#include <stdio.h>

int main(void) {
	int i;                   // 4행
	scanf("%d", &i);         // 5행
	printf("i is %d", i);    // 6행
}
```

- 4행
    - 정수형 변수 i가 선언된다
    - 이 i에 실제 정수 값을 저장하고자 한다.
- 5행
    - 입력을 받는 `scanf` 함수의 경우 변수의 주소값을 전달해야하므로 변수명 앞에 `&` 기호를 붙여줘야 한다.
    - `scanf("%d", &i)` 를 통해 사용자가 입력하는 정수를 i가 저장된 주소에 저장한다.
    - `%d` 는 정수 값을 담아두는 **플레이스 홀더**이고, `&i` 는 i라는 **변수의 주소**를 뜻한다.
- 6행
    - 변수의 값을 화면에 보여준다.

# Topic5: 컴퓨터로 수식을 연산할 수 있을까?

## 핵심 단어
- 연산자
- 산술 연산자
- 할당 연산자

## 수업 내용

### 1. 연산자

C에서는 덧셈, 뺄셈, 곱셈, 나눗셈 같은 기본 산술 연산 뿐만 아니라 나눗셈 후 생기는 나머지의 값을 찾아주는 연산도 사용할 수 있다.

### 2. 산술 연산자
```c
int a = 2+8;
int b = 10-3;
int c = 4*7;
int d = 10/2;
int e = 10/3; // 10/3의 결과는 실수지만, 정수형 변수인 e에는 소수점 이하의 값을 제외한 정수 값만 담긴다.
int f = 13%3;
```

- C의 산술 연산자는 **수학적 기능**을 한다.
- int형 변수와 나눗셈을 같이 수행해줄 때, 정수는 정수가 아닌 변수를 저장하지 못한다는 것을 알고 있어야 한다.

### 3. 할당 연산자
```c
e = f+1; // e의 값: 2
e = e+1; // 3
e += 1; // 4
e *= 7; // 28
e++; // 29
```

- C는 **변수의 값을 갱신**할 수 있는 할당 연산자도 사용 가능하다.
- 변수명 뒤에 ++나 —를 붙여서 변수의 값을 1만큼 증가 또는 감소시켜주는 문법이 있다. 이와 같은 연산자를 **증감 연산자**라고 한다.

# Topic6: 선택할 사항이 생길 때 프로그램으로 어떻게 구현할까? 

## 핵심 단어
- 조건
- 부울 연산식
- if문
- switch문

## 수업 내용
### 1. 부울 연산식과 조건문

- **특정 코드가 특정 상황에서만 실행되도록** 의사 결정하는 코드를 만들 때 조건문을 사용
- 조건문은 대게 어떠한 값이 참인지 거짓인지를 나타내는 **부울 연산식**을 통해 동작

### 2. 부울 연산자
- 참과 거짓을 판단하는 부울 연산식을 만드는 데 사용됨
- **<. >, ==. <=, >=, !=** 가 부울 연산자에 해당됨

### 3. 조건문
- 조건 분기: 다른 상황에 따라 다른 코드가 실행되어야 한다는 개념
1. if - else문
    - if문: if 조건이 참일 때 코드가 실행됨
    - else문: if 조건이 거짓일 때 실행됨

    ```c
    if (x > 0)
    {
    	printf("positive\n"); // x>0이면 이 코드가 실행됨
    }
    else if (x < 0)
    {
    	printf("negative\n"); // x<0이면 이 코드가 실행됨
    }
    else
    {
    	printf("zero\n"); // x=0이면 이 코드가 실행됨
    }
    ```

2. switch문

    조건식의 결과값에 따라 매칭되는 case의 코드가 실행됨

    ```c
    switch (x)
    {
    	case 1:
    		printf("A\n"); // x==1이면 이 코드가 실행된 후, break에 의해 switch문을 빠져나옴
    			break;
    	case 2:
    		printf("B\n"); // x==2이면 이 코드가 실행된 후, break에 의해 switch문을 빠져나옴
    		break;
    	default:
    		printf("C\n"); // x가 1, 2 모두에 해당되지 않으면 이 코드가 실행됨
    }
    ```

3. 3항 연산자

    식이 참이면 : 기호 왼편의 값으로 계산되고, 거짓이면 오른편의 값으로 계산됨

    ```c
    int y = (x > 3) ? 2 : 1; // x>3이 참이면 y는 2가 되고, 그렇지 않으면 y는 1이 된다.
    ```

# Topic7: 같은 코드를 여러 번 작성하지 않기 위해 무엇을 이용하면 좋을까?

## 핵심 단어
- 반복문
- for 반복
- while 반복
- 무한 반복
- do-while 반복

## 수업 내용
### 1. 반복문
동일한 코드를 여러 번 반복하는 방법

### 2. for문
- 반복을 몇 번 수행해야 하는지 정해져 있을 때 사용하는 반복문
- for문을 정의하기 위해선 **(초기식; 조건식; 증감식)** 세 가지 부분이 필요

    ```c
    for(int i=0; i<10; i++) {
    	prinf("hello!\n");
    }
    ```

### 3. while문
- 특정한 조건을 만족하는 동안 실행되는 반복문
- 얼마나 많이 반복되어야 하는지 사전에 알지 못할 때 유용하게 쓰임

    ```c
    int k=0;
    while(k<10) {
    	printf("%d\n", k);
    	k++;
    }
    ```

### 4. do-while문
- while문에 붙어있는 조건이 거짓이 될 때까지 반복된다는 점에서 while문과 비슷함
- 하지만, do-while문은 **조건에 상관없이 항상 최소 한 번은 실행한 후 조건을 확인**한다.

    ```c
    int j;
    prinf("Positive Number: ");
    do {
    	scanf("%d", &j); // 조건에 상관없이 해당 코드를 실행한 후, 아래 줄에서 조건을 확인
    } while(j<=0); // 이 코드에서 조건이 만족되면 다시 반복 실행, 그렇지 않으면 반복문 종료
    ```
